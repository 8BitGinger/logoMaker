SVG 4 Free

This is a simple App using Node.js to generate a basic Logo as an SVG file

How to Use!

  • Start by Cloning the Repo
    • VIEW REPO
  • Open the Repo using VS Code
  • Right-Click the file
    index.js
    and Open in Integrated Terminal
  • Install Dependancies
    1. Node.js
    2. Jest
    3. Inquirer
    4. Graceful-FS
  • Type the following in the terminal:
    node index.js
    and press Enter
  • Follow the Prompts
  • Your personalized Logo will be saved as:
    logo.svg
